# ReactNativeWeb

借用react-web实现三端（IOS/Android/Web）共用一套代码，生成原生的IOS/Android应用，及web手机应用（html/js）。

MAC系统上

版本：
npm 3+以上

"babel": "^6.5.2",

"react": "^0.14.7",

"react-dom": "^0.14.7",

"react-native": "^0.17.0",

"react-web": "^0.1.3"


安装依赖命令

npm i 

开启web服务命令 

npm run web-dev

打包web页面js文件命令

npm run web-bundle
